# Support

## Public repository issues

Use the target repository's issue templates for reproducible bugs, approved
feature requests, and documentation problems.

Include:

- the affected repository, page, endpoint, or component;
- expected and actual behavior;
- steps to reproduce;
- relevant environment or version information;
- non-sensitive screenshots or logs.

## Project and service enquiries

For product engineering, cloud, AI, cybersecurity, e-commerce, managed IT, or
training enquiries:

- Website: https://trustcodesystem.tech/contact
- Email: hello@trustcodesystem.tech

## Security and private information

Do not include passwords, API keys, private client data, personal information,
or vulnerability details in a public issue. Follow
[SECURITY.md](./SECURITY.md) for private vulnerability reporting.
