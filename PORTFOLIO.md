## Product principles

- **Proof over claims:** public links, measurable outcomes, and inspectable code
  where client confidentiality permits.
- **Security by default:** access control, input validation, secret management,
  dependency hygiene, and security review before release.
- **Visible delivery:** staging links, weekly demonstrations, and written
  decisions.
- **Operational ownership:** monitoring, documentation, incident handling, and
  support after launch.
